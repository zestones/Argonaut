Resolves an arithmetic operation between two operands and returns the resulting value.

This function evaluates the specified arithmetic operation between two operands (left and right). The operands can either be integers or real (floating-point) numbers. It performs addition, subtraction, multiplication, or division based on the specified operation.

The function performs the following checks:

  • Division by zero is detected, and an error is reported.
  • Both operands must be initialized before performing the operation. If any operand is uninitialized, an error is reported.
Parameters
operationThe type of arithmetic operation to perform (e.g., addition, subtraction, etc.).
leftThe left operand of the arithmetic operation.
rightThe right operand of the arithmetic operation.
Returns
A vm_cell containing the result of the arithmetic operation. If the operation type or operand types are invalid, no result is returned, but the program exits.
