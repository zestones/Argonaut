void ydebug ( int  debug)

Prints internal tables and the AST to standard output in verbose mode.

When debugging is enabled, this function prints the contents of key data tables and the Abstract Syntax Tree (AST) to the console. It is useful for verbose debugging during development.

Parameters

void ylog ( int  log)

Logs internal tables to a file if logging is enabled.

When logging is activated, this function writes the contents of various internal tables to a file named log.txt. It helps with debugging or reviewing the internal state of the interpreter/compiler after a run.

Parameters
logA flag to indicate whether logging is enabled (non-zero) or disabled (zero).
