Validates the number of arguments provided in a function call.

This function checks if the number of arguments provided in a function call matches the number of parameters expected by the function. If the number of arguments does not match, an error is thrown.

Parameters
argument_listThe argument list provided in the function call.
parameter_countThe number of parameters expected by the function.

Validates the type of each argument in a function or procedure call.

This function iterates over the provided arguments and validates their types against the expected parameter types for a function or procedure. It logs an error for each type mismatch encountered.

Parameters
argument_listA pointer to the Node representing the list of arguments.
parameter_countThe total number of expected parameters for the function or procedure.
index_lexeme_lexicographicThe lexeme index of the function or procedure.
natureThe nature of the callable (e.g., TYPE_FUNC for functions, TYPE_PROC for procedures).
