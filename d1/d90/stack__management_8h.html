Parameters

void fprintf_vm_stack ( FILE *  out)

Prints the contents of the execution stack.

This function prints the details of all stack frames currently on the execution stack. It prints each stack frame's static link, dynamic link, region index, and the values of the cells in the stack. The frames are printed in a tabular format for easy readability.
