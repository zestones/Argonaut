int resolve_expression_type ( Node *  expression)

Resolves the type of an expression.

This function resolves the type of various types of expressions, including literals, operations, array accesses, and function/procedure calls. It ensures type compatibility for expressions involving operations.

Parameters
expressionA pointer to the node representing the expression to resolve.
Returns
The resolved type of the expression, or NULL_VALUE if an error occurs.
Note
This function handles the resolution of expression types, including operator operands and function calls.
Warning
Type errors or unresolved expressions may lead to semantic errors during type resolution.
