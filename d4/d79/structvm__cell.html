Detailed Description

Struct representing a cell in the virtual machine.

A vm_cell holds a value of a specific type (as indicated by the type field) and includes the value itself stored in a union. Additionally, it tracks whether the value has been initialized or not.

The value field is a union that can hold different types of data (integer, real, boolean, character, or string). The type field indicates which data type is currently stored, and the is_initialized field marks whether the cell contains a valid value.
